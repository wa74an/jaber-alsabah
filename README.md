# Jaber AlSabah — Official Website

Built by Beult · beult.net

## Setup

npm install
npm run dev

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Set domain: jab3ralsabah.com

## Swapping Images

Drop your images into /public/images/ with these exact filenames:

- podium.jpg
- ontrack.jpg
- cockpit.jpg
- garage.jpg
- racing.jpg

## Updating Content

All copy lives in /data/content.js
